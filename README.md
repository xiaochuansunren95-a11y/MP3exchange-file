# 🎵 youtube→mp3 変換アプリ

動画サイトのURLから音声を抽出し、直感的な操作でトリミングや音量の自動均一化（ノーマライズ）を行ってMP3ファイルとしてダウンロードできるWebアプリケーションです。

大学のPBL（課題解決型学習）演習の一環として開発されました。

---

## ✨ 主な機能

1. **簡単URL読み込み**: YouTube等の動画URLを入力するだけで、タイトルや動画の長さを自動で取得・表示します。
2. **直感的なシークバー（トリミング）**: スライダー（レンジスライダー）を用いて、直感的に音声の切り出し範囲（開始・終了時間）を指定できます。
3. **音量の自動均一化（ラウドネス正規化）**: 
   - 異なる動画から抽出した音声のバラつきを解消するため、放送・配信向けの国際標準規格（EBU R128、目標値: -16 LUFS）に準拠した音量均一化処理を自動で行います。
4. **MP3ダウンロード**: 変換完了後、ブラウザ経由で即座に端末へ保存できます。

---

## 🛠 使用技術・ライブラリ

- **Language**: Python 3.x
- **Frontend / UI**: [Streamlit](https://streamlit.io/) (Webアプリケーションフレームワーク)
- **Core Logic**: [yt-dlp](https://github.com/yt-dlp/yt-dlp) (動画・音声ストリーム抽出)
- **Audio Processing**: [FFmpeg](https://ffmpeg.org/) (トリミング、音量均一化、MP3エンコード)

---

## 🚀 開発環境の構築・実行方法

### 1. 前提条件（外部ツールのインストール）
本アプリケーションの動作には **FFmpeg** が必須です。お使いの環境に合わせてインストールし、環境変数（PATH）を通してください。

- **Mac (Homebrew)**: `brew install ffmpeg`
- **Windows (Winget)**: `winget install Gyan.FFmpeg`

### 2. Pythonライブラリのインストール
リポジトリをクローン後、必要な依存ライブラリをインストールします。

```bash
pip install streamlit yt-dlp

```

### 3. アプリケーションの起動

以下のコマンドを実行すると、ローカルサーバーが起動し、自動的にブラウザでアプリケーション画面（ `http://localhost:8501` ）が開きます。

```bash
streamlit run main.py

```
